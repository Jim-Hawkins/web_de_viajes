CONSIDERACIONES SOBRE NUESTRA PRÁCTICA:

 1-.Para el correcto funcionamiento de las cookies se necesitará que se abra el archivo html en el navegador de Mozilla Firefox.

 2-.Nuestra consideración para las funcionalidades de javascript ha sido crear un archivo .js para cada una de las funcionalidades que nuestra
 página requería. Estas funcionalidades son:
    - El uso de cookies. En el archivo cookies.js definimos las funciones que son requeridas para la utilización de cookies con el objetivo de
    almacenar los datos del usuario.
    - La funcionalidad del drag & drop. El archivo dragdrop.js contendrá las funciones necesarias para obtenerlo.
    - El buscador de experiencias. Con el archivo searchBar conseguiremos esta funcionalidad.
    - Las opciones del usuario. En el archivo user_options.js definimos las funciones para conseguir las opciones del usuario
    necesarias tales como cerrar sesión, mi perfil mis experiencias...
